# password-generator

A fast gassword generator.

## Installation
Install cargo if you haven't:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
On Windows, use:
```PowerShell
choco install rust

##Usage
Clone:
```bash
cargo run
```

Build:
```bash
cargo build 
```

Run:
```bash
cargo run
```
